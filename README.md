# 🔥 Deals Hub 🔥

A simple website to showcase trending deals from **Amazon, Flipkart, and Meesho** with product images, names, and direct links.

---

## 📌 Features
- Clean and responsive design
- Deal cards with product image, title, and link
- Easy to extend with more shops or products
- Static demo version (works without backend)
- Can be upgraded to fetch **real deals via Affiliate APIs**
